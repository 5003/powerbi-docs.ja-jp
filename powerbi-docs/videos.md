# <a name="power-bi-videos"></a>Power BI のビデオ
Power BI 用の YouTube チャンネルがあります。 [弊社のチャネル](https://www.youtube.com/user/mspowerbi/videos)にアクセスできます。Power BI の初心者の場合は、「[Power BI プレイリストを使用したデータの分析と仮想化](https://www.youtube.com/playlist?list=PL1N57mwBHtN0JFoKSR0n-tBkUJHeMP2cP)」から始めるのがよいでしょう。

* このビデオでは、Power BI のデータを最適化するためのいくつかのヒントについて説明します。 サービスにアップロードする前に、データの書式を設定する最善の方法について説明します。
  
  <iframe width="500" height="281" src="https://www.youtube.com/embed/l2wy4XgQIu0" frameborder="0" allowfullscreen></iframe>
* Power BI を使用すれば、ローカル コンピューターからのファイルのアップロードが迅速で簡単です。 その方法について説明します。
  
  <iframe width="500" height="281" src="https://www.youtube.com/embed/ETj-z3NyY_o" frameborder="0" allowfullscreen></iframe>
* OneDrive を使用することは、データに接続し、常にデータが最新の状態になっていることを確認するための優れた方法です。
  
  <iframe width="500" height="281" src="https://www.youtube.com/embed/6cOjiV_klrs" frameborder="0" allowfullscreen></iframe>
