# <a name="reorder-pages-in-a-report-in-power-bi"></a>Power BI のレポートでページを並べ替える
Power BI では、レポートは 1 つまたは複数のページで構成されます。  各ページの下部にはタブがあります。  レポートの順序を変更するには、タブを選択し、新しい位置にドラッグするだけです。
