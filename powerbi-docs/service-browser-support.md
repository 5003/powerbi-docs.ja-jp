# <a name="supported-browsers-for-power-bi"></a>Power BI のサポートされているブラウザー
Power BI は、次のブラウザーで実行されます。

* Microsoft Edge
* Internet Explorer 11
* Chrome デスクトップの最新バージョン
* Safari Mac の最新バージョン
* Firefox デスクトップの最新バージョン

これらのブラウザーは、使用可能なすべてのプラットフォームでサポートされています。
